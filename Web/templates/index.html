Strikethrough: An interactive tool to visualize the fault-tolerance of various error control code algorithms.

This tool is meant to serve as a demonstration of the performances of various linear error control code algorithms. Tradeoffs in encryption/decryption times, accuracy, and corrections are all accounted for to help users build an intuition for application-specific use cases of each algorithm.
